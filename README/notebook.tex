    \subsection{初始化参数}\label{ux521dux59cbux5316ux53c2ux6570}

在使用mxnet时，要记住，一个网络在使用前必须先初始化

    \begin{Verbatim}[commandchars=\\\{\}]
{\color{incolor}In [{\color{incolor} }]:} \PY{n}{net} \PY{o}{=} \PY{n}{Net}\PY{p}{(}\PY{p}{)}
        \PY{c+c1}{\PYZsh{} Initialize on CPU. Replace with `mx.gpu(0)`, or }
        \PY{c+c1}{\PYZsh{} `[mx.gpu(0), mx.gpu(1)]`,etc to use one or more GPUs.}
        \PY{n}{net}\PY{o}{.}\PY{n}{initialize}\PY{p}{(}\PY{n}{mx}\PY{o}{.}\PY{n}{init}\PY{o}{.}\PY{n}{Xavier}\PY{p}{(}\PY{p}{)}\PY{p}{,} \PY{n}{ctx}\PY{o}{=}\PY{n}{mx}\PY{o}{.}\PY{n}{cpu}\PY{p}{(}\PY{p}{)}\PY{p}{)}
        \PY{c+c1}{\PYZsh{} 还有一种初始化网络的方法，有时候我们需要对不同的网络层采用不一样的}
        \PY{c+c1}{\PYZsh{} 初始化方式，我们可以在网络定义时使用`weight\PYZus{}initializer`这个参}
        \PY{c+c1}{\PYZsh{} 数来指定不同的初始化方式，然后在生成对象时，直接使用`net.initialize()`}
        \PY{c+c1}{\PYZsh{} 来进行初始化，具体可以参考下面这个例子}
\end{Verbatim}


    An example:

\begin{verbatim}
class Net(gluon.Block):
    def __init__(self, **kwargs):  # 可以传入参量控制网络的参数
        super(Net, self).__init__(**kwargs)
        with self.name_scope():
            # conv1 使用了正态分布的初始化方式
            self.conv1 = nn.Conv2D(6, kernel_size=5, weight_initializer=mx.init.Normal())
            self.pool1 = nn.MaxPool2D(pool_size=(2,2))
            # fc1 使用了Xavier的初始化方式
            self.fc1 = nn.Dense(120,weight_initializer=mx.init.Xavier())

    def forward(self, x):
        x = self.pool1(F.relu(self.conv1(x)))
        x = x.reshape((0, -1))
        x = F.relu(self.fc1(x))
        return x
net = Net()
# 由于网络定义时已经定义了初始化参数的方式，这里就不需要指定初始化参数的方式了
net.initialize(ctx=mx.cpu())
\end{verbatim}
